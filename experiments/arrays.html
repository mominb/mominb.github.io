Arrays and how to loop them

An array is like a pile of data. Just like a pile of books. So I learned how to find something from a pile of things.

As an example I choose to find a given type of tags that have source attribute in any web page. Like finding all the urls in a page.

Logic

  1. Overall goal is to write a function that print the src attribute of any tags like img, video or script.
  2. First find all tags with a given name
  3. Check all tags that are returned by previous statement if any one of them has src attribute.
  4. If src attribute is present then print the src attribute on console.

Code

function printTags(tagName) {
    var tags = document.getElementsByTagName(tagName)
    for (var i = 0; i < tags.length; i++) {
        var tag = tags[i];
        if (tag.src) {
            console.log(tag.src)
        }
    }
}
