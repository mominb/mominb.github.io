Arrays and how to loop them

Array is like a pile of data. Just like a pile of books. So I learned how to find something from a pile of things.

As an example I choose to find a type of tags in any web page.

Logic

  1. Write a function that print all tags with a name and src attribute.
  2. First find all tags with a name
  3. Check all tags that are returned by previous statement if it has src attribute.
  4. If src attribute is present then print the src attribute on console.

Code

function printTags(tagName) {
    var tags = document.getElementsByTagName(tagName)
    for (var i = 0; i < tags.length; i++) {
        var tag = tags[i];
        if (tag.src) {
            console.log(tag.src)
        }
    }
}
